HTTP status Codes:-
HTTP response status code are used to specify or indicate whether a particular http req successfully completed or not.

1.informational resposne code(100-199)
2.Successful response(200-299)
3.Redirection message(300-399)
4.Client error response(400-499) ex:- 404
5.Server error response(500-599)

Common real-world meaning:-
200 → “Everything worked”
404 → “Wrong address”
401 → “You need to log in”
403 → “You’re not allowed”
500 → “Server broke”
